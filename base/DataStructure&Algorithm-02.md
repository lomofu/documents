#### 跳表查询的时间复杂度分析

n/2、n/4、n/8、第k级索引结点的个数是n/(2^k)

假设索引有h级，最高级的索引有2个结点。n/(2^h) = 2,从而求得h = log2(n)-1,所以复杂度为 $O(log (n))$
